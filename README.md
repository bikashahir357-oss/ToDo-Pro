# 📝 ToDo Pro

A modern and responsive **To-Do List Web Application** built with **HTML, CSS, and JavaScript**.

It helps users manage daily tasks with features like task priority, due dates, search, filtering, dark mode, and local storage.

## ✨ Features

- ✅ Add Tasks
- ✏️ Edit Tasks
- 🗑️ Delete Tasks
- ✔️ Mark Tasks as Completed
- 💾 Local Storage Support
- 🔍 Live Search
- 🎯 Filter (All / Pending / Completed)
- ⭐ Task Priority (High / Medium / Low)
- 📅 Due Date
- 📊 Dashboard Statistics
- 📈 Progress Bar
- 🌙 Dark / Light Mode
- 📤 Export Tasks (JSON)
- 📥 Import Tasks (JSON)
- 🧹 Clear Completed Tasks
- 🗑️ Delete All Tasks
- 📱 Fully Responsive Design
- ✨ Modern Glassmorphism UI

## 📥 Installation

1. Clone the repository

```bash
git clone https://github.com/bikashahir357-oss/ToDo-Pro.git
```

2. Open the project folder

```bash
cd ToDo-Pro
```

3. Open `index.html` in your browser.
